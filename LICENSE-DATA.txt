--------------------------------------------------------------------
OFFLINE SNAPSHOT NOTICE
--------------------------------------------------------------------
The bundled file data/snapshot_ocds.json is SYNTHETIC sample data shaped
as OCDS v1.1 releases. It exists only to let the system run offline / in
demo mode when the live API is unreachable. Supplier names are real company
names used illustratively; the award amounts, dates, and statuses are
GENERATED and do NOT represent real contracts. For real analysis, run
"Sync Now" while online to ingest actual Find a Tender data.
